### Mini Exchange Engine

A simplified, in-memory matching engine built in pure Java to understand how financial exchanges process and match orders using price-time priority.

#### This project is designed as a learning-focused implementation of core exchange mechanics such as:

- Limit order matching
- Order book management
- Deterministic event processing
- Single-threaded matching architecture
- FIFO price-time priority

## 🚀 Project Goal
To build a deterministic, event-driven matching engine similar in structure to those used in trading systems — but simplified for educational purposes.
This project focuses only on the core matching logic, not:
- User accounts
- Risk management
- Databases
- Distributed clustering
- Production-grade networking
